## ChromaClient

The main client for interacting with Chroma. Manages collections and provides access to the database.

### Constructor

```typescript
new ChromaClient(params?: ChromaClientParams)
```

**Parameters:**

<ParamField path="params" type="ChromaClientParams" optional>
  Client configuration options
  
  <Expandable title="ChromaClientParams properties">
    <ParamField path="path" type="string" default="http://localhost:8000">
      The base URL for the Chroma API server
    </ParamField>
    
    <ParamField path="fetchOptions" type="RequestInit" optional>
      Additional fetch options for HTTP requests
    </ParamField>
    
    <ParamField path="auth" type="AuthOptions" optional>
      Authentication configuration
    </ParamField>
    
    <ParamField path="tenant" type="string" default="default_tenant">
      The tenant to use
    </ParamField>
    
    <ParamField path="database" type="string" default="default_database">
      The database to use
    </ParamField>
  </Expandable>
</ParamField>

**Example:**

```typescript
const client = new ChromaClient({
  path: "http://localhost:8000"
});
```

### Collection Methods

#### createCollection

Creates a new collection with the specified properties.

```typescript
await client.createCollection(params: CreateCollectionParams): Promise<Collection>
```

**Parameters:**

<ParamField path="params" type="CreateCollectionParams" required>
  <Expandable title="CreateCollectionParams properties">
    <ParamField path="name" type="string" required>
      The name of the collection
    </ParamField>
    
    <ParamField path="metadata" type="CollectionMetadata" optional>
      Optional metadata associated with the collection
    </ParamField>
    
    <ParamField path="embeddingFunction" type="IEmbeddingFunction" optional>
      Custom embedding function for the collection. Defaults to DefaultEmbeddingFunction
    </ParamField>
    
    <ParamField path="configuration" type="CreateCollectionConfiguration" optional>
      Collection configuration options
    </ParamField>
  </Expandable>
</ParamField>

**Returns:** Promise resolving to the created Collection instance

**Throws:** 
- `ChromaConnectionError` - If unable to connect to the server
- `ChromaServerError` - If there is an issue creating the collection

**Example:**

```typescript
const collection = await client.createCollection({
  name: "my_collection",
  metadata: {
    "description": "My first collection"
  }
});
```

#### getCollection

Gets an existing collection with the specified name.

```typescript
await client.getCollection(params: GetCollectionParams): Promise<Collection>
```

**Parameters:**

<ParamField path="params" type="GetCollectionParams" required>

    <ParamField path="embeddingFunction" type="IEmbeddingFunction" optional>

### Utility Methods

#### version

Returns the version of the Chroma API.

```typescript

```

**Returns:** Promise resolving to the version of the Chroma API

**Throws:** `ChromaConnectionError` if unable to connect to the server

**Example:**

```typescript

```

#### heartbeat

Returns a heartbeat from the Chroma API.

```typescript
await client.heartbeat(): Promise<number>
```

**Returns:** Promise resolving to the heartbeat in nanoseconds

**Throws:** `ChromaConnectionError` if unable to connect to the server

**Example:**

```typescript
const heartbeat = await client.heartbeat();
```

#### reset

Resets the state of the database.

```typescript
await client.reset(): Promise<boolean>
```

**Returns:** Promise that resolves when the reset operation is complete

**Throws:** 
- `ChromaConnectionError` - If unable to connect to the server
- `ChromaServerError` - If the server experienced an error

**Example:**

```typescript
await client.reset();
```

## CloudClient

A client for connecting to Chroma Cloud. Extends ChromaClient with cloud-specific configuration.

### Constructor

```typescript
new CloudClient(params: CloudClientParams)
```

**Parameters:**

<ParamField path="params" type="CloudClientParams" required>
  <Expandable title="CloudClientParams properties">
    <ParamField path="apiKey" type="string" optional>
      Your Chroma Cloud API key. If not provided, uses the `CHROMA_API_KEY` environment variable
    </ParamField>
    
    <ParamField path="database" type="string" optional>
      The database to use
    </ParamField>
    
    <ParamField path="tenant" type="string" optional>
      The tenant to use
    </ParamField>
    
    <ParamField path="cloudHost" type="string" default="https://api.trychroma.com">
      The Chroma Cloud host URL
    </ParamField>
    
    <ParamField path="cloudPort" type="string" default="8000">
      The Chroma Cloud port
    </ParamField>
  </Expandable>
</ParamField>

**Throws:** `Error` if no API key is provided and `CHROMA_API_KEY` environment variable is not set

**Example:**

```typescript
const client = new CloudClient({
  apiKey: "your-api-key",
  database: "my_database"
});
```

<Note>
CloudClient inherits all methods from ChromaClient. See the ChromaClient documentation above for available methods.
</Note>
